# 🏗️ DevArchitectAI

> Instantly turn your raw app idea into a structured MVP development blueprint.

## ✨ Features
* **Tailored Tech Stack:** AI-recommended tools for your specific idea.
* **Visual Folder Structure:** Clean ASCII tree representation ready to copy.
* **Implementation Plan:** Step-by-step phased tasks to build your MVP.

## 🛠️ Built With
React.js, Vite, Tailwind CSS v4, and Google Gemini 2.5 Flash API.
